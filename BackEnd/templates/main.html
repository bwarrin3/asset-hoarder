Welcome to the Asset Hoarder API

This API allows you to interact with an SQL database containing assets such as images, audio, and video. Below are the available endpoints for retrieving and managing asset data.

1. Get Assets

Retrieve all assets stored in the database.

Quick Link: /data/assets

2. Get Image Assets

Retrieve image assets, optionally filtered by a specific tag.

Quick Link: /data/image_assets

3. Get Audio Assets

Retrieve audio assets, optionally filtered by a specific tag.

Quick Link: /data/audio_assets

4. Get Video Assets

Retrieve video assets, optionally filtered by a specific tag.

Quick Link: /data/video_assets

5. Create Account

Create a new user account by providing a username and password.

Quick Link: /data/create_account

6. User Login

Log in to the system by providing a valid username and password.

Quick Link: /data/login
