Welcome to the Asset Hoarder API

This API allows you to interact with an SQL database containing assets such as images, audio, and video. Below are the available endpoints for retrieving and managing asset data. POST quick links work in progress

1. Get Assets - GET

Retrieve all assets stored in the database.

Quick Link: /data/assets

2. Create Account - POST

Create a new user account by providing a username and password.

Quick Link: /data/create_account

3. User Login - POST

Log in to the system, getting auth cookie, by providing a valid username and password.

Quick Link: /data/login

4. User Logout - GET

Logout of the system removing auth cookie

Quick Link: /data/logout

5. User Toggle Save Asset - POST

Save or unsave provided asset to user if logged in.

Quick Link: /data/user_toggle_save_asset/<int:asset_id>

6. User Toggle Save Asset - GET

View saved assets to user if logged in.

Quick Link: /data/user_saved_assets
